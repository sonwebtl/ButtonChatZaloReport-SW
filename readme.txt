== Description ==
display Chat Zalo ,call mobile,report click from chat zalo,setting change color zalo
fixed chat zalo location left
=== From within WordPress ===

1. Visit 'Plugins > Add New'
2. Search for 'Contact Zalo Report SW'
3. Activate Button Contact Zalo Report SW from your Plugins page.

=== Manually ===

1. Upload the 'Contact Zalo Report' folder to the '/wp-content/plugins/' directory
2. Activate the Chat Zalo display plugin through the 'Plugins' menu in WordPress
3. Go to "after activation" below.

== Installation ==
1. Visit \Plugins > Add New
2. Search for \Contact Zalo Report SW\
3. Activate Contact Zalo Report from your Plugins page.
